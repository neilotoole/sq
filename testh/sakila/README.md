# `testh/sakila`

Test constants for the Sakila test sources: the `@sakila_*` source handles,
handle-set helpers (`SQLEmbedded`, `SQLAll`, `SQLLatest`, `CrossSourceDests`,
…), and dataset facts (table names and row counts). See
[`sakila.go`](./sakila.go).

For the full picture — what Sakila is, the [`sakiladb`](https://github.com/sakiladb)
Docker images, embedded vs external sources, the engine/version matrix, and how
Sakila is used across the repo — see [`docs/SAKILA.md`](../../docs/SAKILA.md).
